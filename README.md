# 🛒 Ecommerce Application

A full-featured Ecommerce application built to simulate an online shopping platform. This project demonstrates backend development concepts including product management, user handling, and order processing.

---

## 🚀 Features

- 👤 User Registration & Authentication  
- 🛍️ Product Listing & Management  
- 🛒 Add to Cart functionality  
- 📦 Order Placement  
- 🔍 Search and filter products  
- 🧾 Order history tracking  

---

## 🛠️ Tech Stack

- **Backend:** Java  
- **Framework:** Spring Boot  
- **Database:** MySQL  
- **Build Tool:** Maven  
- **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/Vatsasarthak/Ecommerce.git
cd Ecommerce
2. Configure Database
Create a MySQL database
Update credentials in application.properties
3. Build the project
mvn clean install
4. Run the application
mvn spring-boot:run
